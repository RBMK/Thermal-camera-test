Thermal camera test that can display all webcam-based thermal cameras and can recalibrate T2-Search camera on demand and only then. It sends 32768 to Focal_length register. 
Other than that, usual OpenCV stuff packed inside. If you want english, run it trough your favorite translator.

This program comes with absolutely no warranty. It can be unpredictable because of coded with Gemini. Try at your own risk for educational purpose or tinkering.
